JourneyAV : Part 3: Publish the MP3

Created by John Stonecypher, last modified on Jan 19, 2026

The procedure for Processing the Sermon Recording consists of three main parts:

  • Part 1: Convert the WAV file to MP3.

  • Part 2: Archive the WAV.

  • Part 3: Publish the MP3.

This page provides detailed instructions for Part 3: Publish the MP3.

Overview

The AV Team uses AWS Simple Storage Service (S3) to host MP3 sermon recordings for online distribution.

The instructions below show you how to publish a new sermon recording to AWS. In this process, you will open S3, find the proper location, upload the MP3, and make it public.

Instructions

To publish a new sermon recording:

  1. Using the AV Laptop, launch the Chrome Internet browser.

Chrome automatically starts on the AWS Sign-In page.

  1. On the AWS Sign-in page, click the IAM username box and choose the dropdown option to autofill the Journey’s AWS Account credentials. 

The rest of the login process completes automatically.

  1. On the Console Home page, launch the S3 service.

  1. On the “Buckets” page, open thejourneyteachings bucket.


  2. In thejourneyteachings page, open the Teachings_August2015_To_Now object.

  1. This location contains all of The Journey’s sermon recordings for the past several years. To add the new recording, choose Upload.

  2. Choose Add Files.

  1. Choose the MP3 file you created in Part 1.

  1. When the file appears in the Upload page, choose Upload.

    The upload process can take 1-2 minutes.

  1. When the upload process completes, choose Close.

    This returns you to the collection of sermon recordings.

  2. To find the file you just uploaded, click the Last Modified column heading twice.

    This sorts the list in date order, bringing your file to the top of the list.

  3. Select your file by clicking the checkbox next to it.


  4. Choose Actions.

    This opens a dropdown menu of options.

  5. From the dropdown, choose Make public using ACL.

    The ACL page displays, asking you to confirm.

  6. Choose Make Public.

  1. A success notification displays.


Publishing is complete. The recording is now available for Internet distribution.

Congratulations!

You have completed the procedure for Processing the Sermon Recording.
