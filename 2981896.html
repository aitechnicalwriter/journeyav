JourneyAV : Part 2: Archive the WAV

Created by John Stonecypher, last modified on Jan 19, 2026

The procedure for Processing the Sermon Recording consists of three main parts:

  • Part 1: Convert the WAV file to MP3.

  • Part 2: Archive the WAV.

  • Part 3: Publish the MP3.

This page provides detailed instructions for Part 2: Archive the WAV.

Overview

The AV Team records each sermon in a high-quality WAV file. We distribute these recordings in the more compact MP3 format, but we maintain an archive of the raw WAV files on the SSD (Solid State Drive).

The instructions below show you how to add a recording to the archive. In this process, you will create an archive folder and cut-and-paste the raw WAV file into that folder.

Instructions

To archive a new recording:

  1. In Windows Explorer, locate the original audio recording on the SSD:

D/AHQU/USBREC/QU-ST001.wav

  1. CUT the file from that location (We will Paste it elsewhere in a moment).

The archive folders are organized by year inside the directory you’re already in (USBREC).

  1. In the directory you’re already in (USBREC), you can see the archive folders, organized by year. Open the folder for the current year.

  1. The [Year] folder contains sub-folders, one for each Sunday of the year. Create a New Folder.

  1. Name the folder for the current date in MM-DD format. For example, a folder for January 18 would be named 01-18.

  1. Open the folder you just created, and Paste the file you copied in Step 1 above.

Archiving is complete.

Congratulations!

In the procedure for Processing the Sermon Recording, you have completed Part 2.
