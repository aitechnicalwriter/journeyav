JourneyAV : Part 1: Convert the WAV file to MP3

Created by John Stonecypher, last modified on Jan 19, 2026

The procedure for Processing the Sermon Recording consists of three main parts:

  • Part 1: Convert the WAV file to MP3.

  • Part 2: Archive the WAV.

  • Part 3: Publish the MP3.

This page provides detailed instructions for Part 1: Convert the WAV file to MP3.

Overview

The AV Team records each sermon in a high-quality WAV file on a Solid-State Hard Drive (SSHD). Before we can distribute a recording on the Internet, we need to convert it to MP3 format.

The instructions below will show you how to convert the raw WAV file to MP3. In this process, you will get the file from the SSHD, set up conversion options, perform the conversion, and add metadata.

Instructions

To convert the file:

  1. Walk to to AV Road Case (next to the Stage), and and detach the SSHD (with cord) from the Mixer.

  2. Bring the SSHD back to the AV table and plug it into the Laptop.

  3. On the Laptop, launch the SwitchPlus application.

  4. On the SwitchPlus main view, choose Add Files.

  5. Open QU-ST001. This is the raw WAV audio file that just got recorded.

  6. When the file appears in the main view, open Options.

The MP3 Output Options popup displays.

  1. In the popup, select the Volume tab.

A “Volume Normalization” process runs automatically.

  1. When the automatic process finishes, choose OK.

  1. Back in the main view, Convert the file.

The conversion process can take up to a minute.

  1. Next, we’ll add metadata to the new MP3 file. Right-click on QU-ST001 and choose Edit Audio Tags of Converted Files.

The Audio Tags Editor pops up.

  1. In the pop-up, enter the following metadata:

    1. Title: Enter the title of the sermon. You can find this in the Service Roadmap.

    2. Artist: Enter the name of the speaker (e.g., “Michael Noel”).

    3. Album Artist: Enter “The Journey Church”.

    4. Album: Enter the title of the sermon series. You can find this in the Service Roadmap.

  1. When the tags are complete, click OK.

  2. Back in the main view, choose Open Output Folder.

  1. In the Sunday Teachings folder, select the QU-ST001.mp3 file you just made.

  1. Rename the file, using the format: YYYY-MM-DD-SpeakerName-SermonTitle. Press Enter to finish.

File conversion is complete.

Congratulations!

In the procedure for Processing the Sermon Recording, you have completed Part 1.

Proceed to Part 2
